# GoTelegram — MTProxy с Fake TLS

Bash-скрипт для быстрого развёртывания MTProxy-сервера на базе [nineseconds/mtg](https://github.com/9seconds/mtg) с маскировкой трафика (Fake TLS). Позволяет стабильно использовать Telegram в условиях ограничений DPI.

## Как это работает

Скрипт разворачивает MTProxy внутри Docker-контейнера. При генерации secret выбирается домен-маскировка: провайдер видит обычный TLS-трафик к, например, `wikipedia.org`, а не проксируемый Telegram.

## Требования

- Linux (Ubuntu / Debian / CentOS / Fedora)
- `sudo`-доступ
- Открытый входящий порт (443 или другой на ваш выбор)

> Docker и `qrencode` устанавливаются автоматически, если отсутствуют.

## Быстрая установка

```bash
wget -O gotelegram.sh https://raw.githubusercontent.com/omichs/gotelegram/main/setup_gotelegram.sh \
  && chmod +x gotelegram.sh \
  && sudo ./gotelegram.sh
```

После первого запуска команда `gotelegram` становится доступна глобально:

```bash
sudo gotelegram
```

## Что происходит при первом запуске

1. Устанавливаются зависимости (Docker, qrencode)
2. Скрипт копируется в `/usr/local/bin/gotelegram`
3. Открывается интерактивное меню управления

## Меню управления

```
╔══════════════════════════════════════╗
║      GoTelegram MTProxy Manager      ║
╚══════════════════════════════════════╝
  Статус: ● Запущен

  1) Установить / Обновить прокси
  2) Показать данные подключения
  3) Удалить прокси
  0) Выход
```

### Установка прокси (пункт 1)

- Выбор домена для Fake TLS из списка (20 вариантов) 
- Выбор порта: `443`, `8443` или произвольный
- Автоматическая проверка занятости порта
- Обновление Docker-образа перед запуском
- Контейнер запускается с `--restart always` (переживает перезагрузку сервера)

### Данные подключения (пункт 2)

Выводит внешний IP, порт, secret и готовую ссылку формата:

```
tg://proxy?server=<IP>&port=<PORT>&secret=<SECRET>
```

А также QR-код для мгновенного подключения с телефона прямо в терминале.

## Технические детали

| Параметр | Значение |
|---|---|
| Движок | [nineseconds/mtg:2](https://hub.docker.com/r/nineseconds/mtg) |
| Режим | `simple-run` с Fake TLS |
| DNS | `1.1.1.1` (Cloudflare) |
| IP-стек | Предпочтение IPv4 |
| Хранение secret | Docker labels (надёжно, без хрупкого парсинга) |
| Перезапуск | `--restart always` |

## Структура файлов

```
setup_gotelegram.sh   # единственный файл — скрипт установки и менеджер
```

После установки:

```
/usr/local/bin/gotelegram   # глобальная команда
```

## Удаление

Через меню (пункт 3) — останавливает и удаляет Docker-контейнер.

Для полного удаления команды из системы:

```bash
sudo rm /usr/local/bin/gotelegram
```
